useR! 2014 póló

A bejegyzést r-projekt írta 2014. június hó 10. napján 12:53-kor

Az idei useR! konferencia szervezői érdekes pályázatot írtak ki (a 4 évvel ezelőtti hagyományok mentén): tervezzük meg a konferencia “reproducible” pólóját! Ez annyit tesz, hogy a póló elején szereplő design-elem (plot) reprodukálható kell, hogy legyen a póló hátuljára nyomtatott R kód segítségével.

A pályázatokat ennek megfelelően GitHub-on várják, ahova eddig 10 pályamű érkezett:

A fentebb bemutatott megoldások részletes leírása és forráskódja elérhető az http://daroczig.github.io/t-shirt oldalon. Egy hazai pályázat kódja:

library(pixmap)
 
## get R logo into a matrix
R <- read.pnm(
    system.file('pictures/logo.ppm',
                package = 'pixmap')[1])
 
## drop colors
R <- addChannels(R)@grey
 
## create a matrix full of "useR! 2014"
T <- matrix(
    strsplit('useR!2014', '')[[1]],
    nrow = nrow(R),
    ncol = ncol(R),
    byrow = TRUE)
 
## remove cells not in the R logo
T[which(R > 0.85)] <- ' '
 
## save to disk
write.table(x = T, # never abbreviate TRUE to T :)
            file = 'tshirtImage.txt',
            row.names = FALSE,
            sep = '',
            col.names = FALSE,
            quote = FALSE)

A konferencia résztvevői online kérdőíven szavazhatnak, hogy melyik pólóterv legyen az idei rendezvény hivatalos arculata.

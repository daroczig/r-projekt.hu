Az oldal szamos izgalmas informaciot tartogat a latogato szamara:

  • az RStudio CRAN mirror letoltesi adataira epulve eleg megbizhato statisztikat szolgaltat a CRAN-en hosztolt R csomagok “nepszerusegerol”,
  • a GitHub csillagok szama alapjan itt is elerheto az ott egyebkent ismert “trending” csomagok listaja
  • es a legfrissebb/legutobb frissitett csomagok sora is elerheto.

Ezeken kivul az oldal tovabbi GitHub-integracioval is buszkelkedhet, peldaul elerhetoek a csomagok korabbi verzio, az azok kozotti diffeket egy kattintassal elerhetjuk, egy helyen latjuk a csomag manualt, vignette-eket, NEWS fajlt es egyebeket is.

Tovabbi orom, hogy az oldal API funkciokkal is bir.
